# Learning Resources

## Purpose

This document lists publicly available documentation and technical resources that supported the recovery process. These references were consulted to verify best practices, troubleshoot issues, and confirm implementation details during various stages of the project.

## Official Documentation

### WordPress

Official documentation for WordPress core, administration, troubleshooting, and maintenance.

https://wordpress.org/documentation/

---

### Cloudflare

Documentation covering DNS management, SSL/TLS configuration, caching, security, and performance.

https://developers.cloudflare.com/

---

### ThemeForest / Envato

Documentation related to purchased themes, licensing, downloads, and account management.

https://help.market.envato.com/

---

## Community Resources

### Internet Archive (Wayback Machine)

Historical snapshots used to compare previous versions of the website and assist with media reconstruction.

### WPBakery Documentation

Reference material for WPBakery Page Builder configuration and troubleshooting.

https://kb.wpbakery.com/

---

### PHP Documentation

Reference for PHP functions, compatibility, and troubleshooting.

https://www.php.net/docs.php

---

### MySQL Documentation

Reference for database management and SQL operations.

https://dev.mysql.com/doc/

## General Engineering Approach

Throughout the recovery process, official documentation was preferred wherever available. Community resources were used to supplement troubleshooting and historical research when required.

The objective was to validate technical decisions using reliable sources while maintaining the integrity of the recovered production environment.
